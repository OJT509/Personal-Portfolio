My Recent Projects

Distributed Ledger Technology

One of my most recent projects is my dissertation, which was based on distributed ledger technology. The idea behind it is was to investigate current voting fraud techniques and how they can be alleviated by proposing a de-centralised, blockchain based voting solution. This was built to run on the Ethereum network in conjunction with Metamask, a Google chrome extension and Ganache which allowed the execution of a locally running blockchain. I am also in the process of building on my old University web-development project which is a CRUD application for cars, implementing features such as advanced validation techniques and OAuth authentication.
