My Recent Project

Twitter Sentiment Analysis

One of my most recent projects was a Twitter Sentiment Analysis Tool created in Python. It allows a user to enter, for example a Twitter handle username and then the application will produce a dataset that you can tailor to what you want to see. E.g. most liked tweet or retweet. It also has the functionality to plot the gathered data on a graph via the MatPlot library for Python, allowing easier visualisation. This tool utilises many libraries such as; Tweepy (A Python library for access to the Twitter API). Inspired from Vincent Russo.
